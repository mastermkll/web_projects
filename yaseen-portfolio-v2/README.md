# Yaseen Shaikh Portfolio

Standalone one-page portfolio website for a professional fitness trainer.

## Files

- `index.html` - page content
- `styles.css` - design and responsive layout
- `script.js` - reveal animation + current year

## Add real media later

1. Create an `assets/` folder in this directory.
2. Add files using these names:
   - `assets/yaseen-hero.jpg`
   - `assets/intro.mp4`
   - `assets/transformation-1.mp4`
   - `assets/transform-1.jpg`
   - `assets/transform-2.jpg`
3. Replace the placeholder cards in `index.html` with `<img>` or `<video>` tags when ready.
